## Installation GitHub Pages

1. Décompresser le ZIP.
2. Uploader `index.html` + les dossiers `css`, `js`, `img`, `assets` à la racine du dépôt.
3. Garder les dossiers, ne pas mettre les CSS en vrac à la racine.
4. Attendre le déploiement GitHub Pages.
5. Recharger le site avec Ctrl + F5.
