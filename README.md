```text
index.html
css/
  base.css
  signal.css
  hero.css
  sections.css
  universe.css
  join.css
  footer.css
  responsive.css
js/
  signal.js
img/
  hero.jpg
  es.png
  logo.png
assets/
  icons/
```

## Installation GitHub Pages

1. Décompresser le ZIP.
2. Sur GitHub, supprimer les anciens fichiers à la racine si besoin.
3. Upload **index.html + les dossiers css, js, img, assets** en conservant la structure.
4. Le fichier `index.html` doit rester à la racine.
5. Les fichiers CSS doivent rester dans `/css`.
6. Les images doivent rester dans `/img`.

Commit conseillé :

```text
V7.0 - Developer Edition project structure
```

## Important

Si la page apparaît sans style, c'est presque toujours que les dossiers `/css` ou `/img` n'ont pas été uploadés au bon endroit.
